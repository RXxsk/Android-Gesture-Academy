```markdown
# Guía de Contribución

¡Gracias por tu interés en contribuir a Android Gesture Academy! 🎉

## 🐛 Reportar Bugs

1. Asegúrate de que el bug no haya sido reportado antes
2. Usa la plantilla de issue correspondiente
3. Incluye:
   - Versión de Android
   - Dispositivo/emulador
   - Pasos para reproducir
   - Comportamiento esperado vs actual

## 💡 Sugerir mejoras

¿Tienes una idea para mejorar la librería? ¡Nos encantaría escucharla!

## 🚀 Enviar Pull Requests

1. Fork el repositorio
2. Crea una rama: `git checkout -b feature/nombre-de-tu-feature`
3. Haz tus cambios
4. Ejecuta las pruebas: `./gradlew test`
5. Commit: `git commit -m 'Añade nueva feature'`
6. Push: `git push origin feature/nombre-de-tu-feature`
7. Abre un Pull Request

## 📝 Estilo de código

- Usamos Kotlin
- Seguimos las guías de estilo de Android
- Mantenemos cobertura de pruebas >80%
```
